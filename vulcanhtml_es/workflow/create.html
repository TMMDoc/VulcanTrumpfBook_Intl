Crear

Crear

Esta página muestra todos los archivos DXF disponibles.

Crear tarea

El usuario debe seleccionar cualquiera de los archivos DXF o varios archivos y, a continuación, seleccionar el botón Crear tarea. Después de crear una tarea, aparecerá una interfaz con pestañas como la que se muestra en la siguiente captura de pantalla. Hay cuatro pestañas en esta ventana, que son las siguientes:

  1. Pestaña Piezas

  2. Pestaña Configuración

  3. Pestaña Chapas

  4. Pestaña Diseños

Pestaña Piezas

En esta, el usuario puede añadir tantas piezas como quiera. Deben seleccionar todos los archivos dxf y seleccionar Crear tarea para añadir todas las piezas. En la pestaña de piezas, el usuario puede cambiar la cantidad máxima, establecer la prioridad, la dirección del grano, la rotación de la pieza y el grupo. En esta pestaña gestionará la lista de piezas (y sus cantidades requeridas) que deben anidarse.

Pestaña Configuración

Le permite modificar parámetros tales como el material, el grosor, la película, el tratamiento y el espacio entre piezas.

Pestaña Chapas

Esta es la pestaña donde gestionará la lista de chapas que se tendrán en cuenta para el nesting. El usuario puede añadir chapas de diferentes tamaños. Aumentar o disminuir la cantidad. El margen también se puede modificar.Utilice TwinLine si necesita un procesamiento eficiente de sus piezas.

Pestaña Diseños

Al seleccionar la pestaña Diseño, Vulcan Control se iniciará automáticamente el cálculo de nesting, si no hay diseños. Una vez realizado el cálculo, presentará los diseños calculados como una lista de miniaturas. Puede editar cualquiera de estos diseños en la ventana principal de Flux haciendo clic en la miniatura. Dispondrá de un completo editor de diseño en el que podrá modificar la posición y orientación de las piezas, editar los útiles, editar la secuencia, etc.

Pulse el botón Recalcular para hacer el nesting del resto (piezas no anidadas). Si desea volver a hacer el nesting desde el principio, elimine los diseños existentes y, a continuación, pulse este botón.

Al seleccionar el botón Editar se abrirá la página de edición del diseño, que se explica detalladamente en la sección 3.3. Al seleccionar el botón Cancelar no se guardarán los cambios realizados aquí.

El usuario puede guardar los diseños utilizando Guardar diseños. Introduzca el nombre y haga clic en Ok para guardar los diseños. El diseño guardado estará disponible en la página de programas.
