Ejecución de la simulación

Para obtener información detallada sobre cómo se definió la secuencia del proceso, puede ejecutar una simulación.

Simulación HMI

En esta página ejecutamos la simulación para el programa. El nombre del programa y su dimensión se muestran en la parte superior. Podemos reproducir, pausar y restablecer la simulación. El deslizador muestra el progreso de la simulación. Se puede arrastrar hacia la izquierda para retroceder o hacia la derecha para avanzar en la simulación.

Este botón se utiliza para ampliar el programa.

Este botón se utiliza para devolver el programa a su posición original.

Este botón se emplea para alejar la vista del programa.

Este botón se utiliza para mover el programa hacia la izquierda.

Este botón se utiliza para mover el programa hacia la derecha.

Este botón se utiliza para mover el programa a la parte superior.

Este botón se utiliza para mover el programa a la parte inferior.

Seleccionar hecho cerrará la ventana de simulación.

Simulación de piezas

También se puede ver con el diodo láser en la operación de prueba láser dónde se cortará la pieza en la chapa. Solo tiene que seleccionar la opción mientras se ejecuta la pieza en modo de prueba láser.
