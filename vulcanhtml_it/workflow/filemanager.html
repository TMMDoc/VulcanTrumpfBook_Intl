Gestione file

Importa

Per importare i file è possibile utilizzare direttamente il pulsante Importa nella parte in alto a destra dello schermo. Consente di importare con un numero minimo di clic.

File manager

Tutte le altre operazioni sui file vengono eseguite utilizzando l’opzione File Manager (disponibile sul lato destro dello schermo).

Le seguenti funzioni sono disponibili nelle operazioni sui file:

  • Rinomina: questo comando consente di modificare il nome del programma selezionato. È possibile digitare il nuovo nome nella finestra di dialogo che viene visualizzata.

  • Cancella: utilizzare questo interruttore per eliminare il programma selezionato.

  • Clona: questo comando consente di creare una copia esatta dei programmi selezionati. Selezionare uno o più programmi e quindi premere questo pulsante per creare una copia del programma selezionato (o dei programmi selezionati). Quando più di un programma viene clonato, si ha una finestra di dialogo Clona file, che chiede se si intende clonare il numero selezionato di file. Una volta premuto il segno OK, i file vengono clonati. Generalmente, il nome il file clonato avrà un numero dopo il nome del pezzo, separato da una tilde (ad es., nel caso in cui si cloni un pezzo denominato P5, il nome del clone sarà P5~1).

  • Esporta: quando viene premuto, questo comando apre la finestra di dialogo Esporta file, attraverso la quale si sceglie la posizione per esportare i file selezionati.

  • Importa: è possibile importare programmi utilizzando questa funzione. Questo pulsante apre la finestra di dialogo Importa file, tramite la quale è possibile selezionare uno o più programmi o i file del programma diretto da importare. Una volta selezionati i file, premere il segno OK nella finestra di dialogo per importare i file. Se il nome del file (o dei file) che si sta cercando di importare già esiste, viene visualizzato un messaggio di avvertimento come mostrato nell’immagine seguente, che chiede se sostituire il file esistente (o i file esistenti). //Questa opzione è presente all’esterno del menu File Manager.

  • Organizza: selezionando l’intestazione del programma nella parte alta viene visualizzata questa opzione. Premendo questo pulsante appare una finestra di dialogo come quella mostrata nella figura in basso.

La pagina Modifica visualizza tutti i programmi della cartella Programmi. È possibile creare cartelle e strutturare i dati all’interno di questa cartella. È possibile creare nuove cartelle utilizzando il comando Nuova cartella. In questo modo si creerà una nuova cartella all’interno della cartella Programmi. A sinistra, si può vedere la sezione Seleziona cartella che mostra la struttura della directory. È possibile scegliere una cartella dalla struttura ad albero. Viene visualizzato il contenuto della cartella selezionata nella pagina Modifica.

È anche possibile disporre i file nella cartella utilizzando le opzioni fornite nella sezione Ordina per.

Per disporre per nome in ordine crescente scegliere Nome [A..Z]. Per ordinare per nome in ordine decrescente scegliere Nome [Z..A].

Il più recente per primo mostra tutti i programmi che sono stati creati di recente, mentre Il più datato per primo mostra i programmi in ordine ascendente di data di creazione.

Utilizzando Mostra icone è possibile attivare o disattivare la visualizzazione delle icone dei programmi nella pagina Modifica. Usare il pulsante Cancella per eliminare una sub-directory.

È possibile eliminare solo una sub-directory vuota.

Premere Fine per applicare le modifiche e uscire.
