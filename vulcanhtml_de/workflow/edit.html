NC-Programm bearbeiten

Mit Vulcan Control haben Sie die volle Kontrolle über Ihr NC-Programm, entweder über Menüschaltflächen oder indem Sie den Text direkt bearbeiten. Wählen Sie das Programm, das Sie bearbeiten möchten, und betätigen Sie dann die Schaltfläche Bearbeiten.

Bearbeitungsseite

  • Rest hinzufügen/löschen: Wenn Sie diese Option wählen, wird eine Resttafel erstellt. Es ist es auch möglich, den Rest zu löschen, indem Sie die gleiche Option wählen, nachdem der Rest bereits erstellt wurde.

  • Anzeige: Der Bediener kann die Tafel- oder Maschinenanzeige ein- oder ausschalten.

  • Microjoint hervorheben: Der Bediener kann sich schnell vergewissern, ob die erforderlichen Konturen mit einem Microjoint ausgestattet sind oder nicht. Wenn Sie diese Option wählen, wird der Microjoint in der Zeichnung hervorgehoben.

  • Einstellungen: Der Bediener kann die Entfernung konfigurieren, was das Hinzufügen von Resten ermöglicht.

  • Abbrechen: Diese Option schließt den Dialog und führt zur Seite Programme zurück, ohne die Änderungen zu speichern.

  • Speichern: Diese Option speichert die vom Bediener vorgenommenen Änderungen.

Wählen Sie das LST-Programm, das Sie bearbeiten möchten, und betätigen Sie dann die Schaltfläche Speichern. Dadurch wird die Seite Bearbeiten angezeigt, auf der der Bediener die erforderlichen Änderungen am Programm vornehmen kann. Auf der LST-Textseite können wir Änderungen am NC-Programm vornehmen. Es stehen die Optionen Suchen Ctrl+F und Ersetzen Ctrl+H eines Texteditors zur Verfügung.

Ausrichtung

Diese Seite zeigt oben den Programmnamen und das Maß an. Der Bediener kann die Ausrichtung, in der das Programm angezeigt werden soll (Hoch- oder Querformat), in den Maschineneinstellungen ändern.
