Produktions-Workflow

In einem typischen Workflow können Sie die folgenden Schritte ausführen:

  • Erstellen Sie eine Tafel aus DXF oder importieren Sie eine aus einem Netzlaufwerk.

  • Wählen Sie das Programm, das Sie ausführen möchten.

  • Überprüfen Sie die Düse.

  • Stellen Sie den Arbeitsversatz ein (definieren Sie die genaue Position der Tafel in der Maschine).

  • Führen Sie das Programm aus.

  • Beladen Sie das nächste Blatt über den Palettenwechsler.

  • Wiederholen Sie den Ablauf.

In diesem Kapitel werden die Details der einzelnen Schritte erläutert.
