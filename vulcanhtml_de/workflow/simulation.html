Simulationslauf

Um detaillierte Informationen darüber zu erhalten, wie der Bearbeitungsreihenfolge definiert wurde, können Sie eine Simulation ausführen.

Bedienoberfläche Simulation

Auf dieser Seite führen wir die Simulation für das Programm aus. Der Programmname und das zugehörige Maß werden oben angezeigt. Wir können die Simulation abspielen, pausieren und zurücksetzen. Der Schieberegler zeigt den Fortschritt der Simulation an. Er kann gezogen nach links gezogen werden, um die Simulation zurückzuspulen, oder nach rechts, um sie vorzuspulen.

Mit dieser Schaltfläche wird das Programm vergrößert.

Mit dieser Schaltfläche wird das Programm in die Ausgangsposition gebracht.

Mit dieser Schaltfläche wird das Programm verkleinert.

Mit dieser Schaltfläche wird das Programm nach links verschoben.

Mit dieser Schaltfläche wird das Programm nach rechts verschoben.

Mit dieser Schaltfläche wird das Programm nach oben verschoben.

Mit dieser Schaltfläche wird das Programm nach unten verschoben.

Wenn Sie fertig wählen, wird das Simulationsfenster geschlossen.

Teilsimulation

Mit der Laserdiode im Lasertestmodus können Sie auch sehen, wo das Teil auf der Tafel geschnitten wird. Wählen Sie einfach die Option aus, während Sie das Teil im Lasertestmodus ausführen.
