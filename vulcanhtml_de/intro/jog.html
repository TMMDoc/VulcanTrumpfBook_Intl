Maschinenbedienung

Der Touchscreen bietet verschiedene Optionen zum Bedienen der Maschine:

Verfahren

Auf dieser Seite kann die Achse verfahren werden. Hier können auch die aktuellen Positionen der X-, Y- und Z-Achse angezeigt werden.

Folgende Verfahrarten stehen zur Verfügung:

  1. Modales Verfahren

  2. Inkrementelles Verfahren

  3. Zielpunktverfahren

Modales Verfahren

Eine länger als 300 ms gedrückte Maus startet eine modale Bewegung, bis der Bediener den Finger hebt/die Maus loslässt (langer Klick). Je nachdem, wo der Bediener auf den Pfeil geklickt hat, bewegt sich die Maschine mit unterschiedlichen Geschwindigkeiten. Die Geschwindigkeit für das modale Verfahren kann in den Einstellungen konfiguriert werden.

Schrittweises Verfahren

Ein Klick (Maus nach unten, innerhalb von 300 ms unmittelbar gefolgt von Maus nach oben) bewegt sie in Schritten. Je nachdem, wo der Bediener auf den Pfeil geklickt hat, bewegt sich die Achse in kurzen, mittleren oder großen Schritten. Die Schrittweite kann in den Einstellungen konfiguriert werden.

Zielpunktfahrt

Der Bediener kann den Zielpunkt der ausgewählten Achse im Eingabefeld angeben. Drücken Sie die START-Taste, um die Bewegung zu starten. Die Bewegung der Achse kann jederzeit mit der STOPP-Taste gestoppt werden.

Geschwindigkeits-Override

Mit Override + und Override - kann der Bediener die Geschwindigkeit von Achsenbewegungen beeinflussen (in Prozent).

  • Override + : Erhöht den Override-Prozentsatz für die konfigurierte/eingestellte Achse oder Fahrbahngeschwindigkeit. Oberhalb von 10 % wird der Wert in Schritten von 10 % geändert, unterhalb von 10 % wird er in Schritten von 1 % geändert.

  • Override - : Senkt den Override-Prozentsatz für die konfigurierte/eingestellte Achse oder Fahrbahngeschwindigkeit. Oberhalb von 10 % wird der Wert in Schritten von 10 % geändert, unterhalb von 10 % wird er in Schritten von 1 % geändert. Der Bediener kann zum Erhöhen oder Senken der Geschwindigkeit auch den Schieberegler verwenden.

Referenzpunktfahrt

Das Anfahren des Referenzpunkts durch die Achsen kann auf zwei Arten erfolgen.

  • Automatisch: Wenn diese Funktion aktiviert ist, wird durch Drücken der Start-Taste das automatische Anfahren des Referenzpunkts eingeleitet. Durch Drücken der Stopp-Taste kann das Anfahren des Referenzpunkts jederzeit abgebrochen werden.

  • Manuell: Wenn diese Funktion ausgewählt ist, kann das manuelle Anfahren des Referenzpunkts durch eine einzelne Achse initiiert werden. Wählen Sie die gewünschte Achse und drücken Sie die Start-Taste, um das Verfahren einzuleiten. Durch Drücken der Stopp-Taste kann das Anfahren des Referenzpunkts jederzeit abgebrochen werden.

Einstellungen

Der Bediener kann die Schrittweite (für schrittweises Verfahren) und die Geschwindigkeit (für modales Verfahren) für die spezifischen Bereiche auf dem Pfeil in den Einstellungen konfigurieren.
