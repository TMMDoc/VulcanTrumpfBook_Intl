Tableau de commande

Le tableau de commande est un élément essentiel pour préparer la machine laser au fonctionnement. Veuillez vous référer au manuel opérateur pour plus de détails.

Le panneau de commande se compose des interrupteurs de commande suivants :

  1. Affichage

  2. Arrêt d’urgence

  3. Mode simulation laser

  4. Touche de validation de l’entraînement

  5. Laser validé

  6. Contacteur de porte

  7. Outrepasser

  8. Port USB

Nom Élément de commande Conditions requises Description

Arrêt d’urgence (bouton coup de poing)

  • L’alimentation électrique du système complet est déconnectée (la tension de commande de 24 V est maintenue).

    • Tous les mouvements d’axe s’arrêtent.

    • Tous les entraînements sont éteints électriquement.

  • L’usinage est interrompu :

    • Le faisceau laser est éteint.

    • Le blocage du faisceau est fermé.

    • L’alimentation en gaz de coupe est interrompue.

    • Un programme d’usinage actif est abandonné et doit être redémarré, si nécessaire.

    • La position initiale de la machine doit être approchée.

    • Les points de référence des axes sont conservés.

  • Le groupe hydraulique, le dépoussiéreur compact et les convoyeurs horizontaux et/ou transversaux, respectivement, sont éteints.

  • Tous les mouvements pneumatiques sont arrêtés ou transférés à un état sans danger. L’alimentation en air comprimé reste inchangée.

  • Il reste du vide.

Machine Marche/Arrêt

Lorsqu’il est enclenché, le bouton clignote.

  • Met la machine en marche :

    • S’allume lorsque la machine opérationnelle.

  • Éteint la machine :

    • S’éteint lorsque la machine est éteinte.

  • L’arrêt de la machine est également déclenché par :

    • Bouton coup de poing d’arrêt d’urgence.

    • Erreurs qui déclenchent un « ARRÊT D’URGENCE »

Vérification laser

  • S’allume si la machine est en mode simulation.

  • Le programme d’usinage sélectionné fonctionnera sans que le faisceau laser ne soit amorcé.

Laser Marche/Arrêt

  • Démarre le cycle de mise en marche automatique du laser :

    • Clignote pendant le cycle de mise en marche.

    • S’allume si le laser est opérationnel.

  • Démarre le cycle de mise à l’arrêt automatique du laser :

    • Clignote pendant le cycle de mise à l’arrêt.

    • S’éteint lorsque la machine est éteinte.

Déverrouillage de la porte

La porte de protection est déverrouillée et peut être ouverte.

Affichage d’état (barrière optique de sécurité interrompue)

L'« ARRÊT D’URGENCE » n’est pas actif, pas d'« arrêt de l’avance » actif.

  • Clignote lorsque la barrière optique de sécurité qui protège le changeur de palettes est interrompue.

  • Réactiver la barrière optique de sécurité : appuyez sur la pédale « Valider la barrière optique de sécurité » au niveau du poteau de démarrage. Appuyez ensuite sur le bouton « Démarrer » sur le tableau de commande. Reprendre les mouvements en pause du changeur de palettes : appuyez sur « Démarrer »

Vitesse d’avance

  • Contrôle la vitesse de déplacement des axes X et Y entre 4 et 120% :

  • Dans l’activité principale de production, le réglage du déplacement est de 4-100 m/min.

Touches de navigation

  • Déplace les axes NC sélectionnés dans la direction (+) ou (-).

    • Déplace les axes dans la direction +/-.

    • Les boutons plus/moins augmentent/réduisent la valeur de « Fonctions manuelles » avec le symbole ±.

Réinitialiser (arrêter)

  • Supprime les messages de dysfonctionnement actifs.

  • Annule un programme actif.

  • Les points de référence sont maintenus.

  • La position initiale peut être perdue.

Arrêt de l’avance (pause)

  • S’allume si « ARRÊT DE L’AVANCE » est déclenché.

    • Tous les mouvements d’axe s’arrêtent.

    • L’usinage est interrompu :

      • Le faisceau laser est éteint.

      • Le blocage du faisceau est fermé.

      • L’alimentation en gaz de coupe est interrompue.

      • Un programme d’usinage actif est mis en pause et peut être exécuté ultérieurement via une validation.

  • Le groupe hydraulique, le dépoussiéreur compact et les convoyeurs horizontaux et/ou transversaux, respectivement, sont éteints.

  • Tous les mouvements pneumatiques sont arrêtés ou transférés à un état sans danger. L’alimentation en air comprimé reste inchangée.

  • Il reste du vide. Pour valider un « arrêt de l’avance », appuyez sur « Validation de l’arrêt de l’avance ».

Validation de l’arrêt de l’avance (lancement de l’avance)

Aucune erreur présente, la barrière optique de sécurité est validée.

  • Clignote si une erreur est présente ou si un « ARRÊT DE L’AVANCE » a été déclenché

  • Par exemple, en interrompant la barrière optique de sécurité, en validant l’erreur active, en validant l'« ARRÊT DE L’AVANCE ».

Démarrer (lancement du programme)

Aucune erreur présente, la machine est opérationnelle.

  • Le bouton « Démarrer » a les fonctions suivantes en fonction de l’activité principale sélectionnée :

    • Si l’activité principale Configuration est sélectionnée : la machine se déplace en position initiale sans commencer à exécuter la liste des ordres ou le programme d’usinage sélectionné.

    • Dans toutes les autres activités principales : la machine se déplace en position initiale et commence à exécuter la liste des ordres.
