Modifier le layout

Vulcan Control prend en charge le format FXLYT de Flux. L’avantage de l’utilisation des fichiers FXLYT est qu’ils sont malléables et l’opérateur a bien plus d’option avec de tels fichiers qu’avec des fichiers DIN durcis.

L’utilisateur doit sélectionner un fichier de layout, puis appuyer sur Modifier pour accéder à la page Modifier le layout. Elle comprend cinq options, comme ci-dessous :

  • Pièce

  • Contour

  • Outillage

  • Squelette

  • Ordre

Pièce

Ajouter : L’utilisateur peut ajouter des pièces DXF dans le layout.

Déplacer : Permet de déplacer la pièce sélectionnée dans le layout.

  1. Déplacer vers la gauche : La pièce se déplace vers la gauche.

  2. Déplacer vers la droite : La pièce se déplace vers la droite.

  3. Monter : La pièce se déplace vers le haut.

  4. Descendre : La pièce se déplace vers le bas.

La grandeur du déplacement de la pièce dépend de la taille de pas sélectionnée.

Rotation : Permet de tourner la pièce sélectionnée dans le layout.

  1. Tourner de 90 degrés vers la droite : La pièce tourne de 90 degrés dans le sens des aiguilles d’une montre.

  2. Tourner de 90 degrés vers la gauche : La pièce tourne de 90 degrés dans le sens inverse des aiguilles d’une montre.

  3. Tourner vers la droite : La pièce tourne dans le sens des aiguilles d’une montre.

  4. Tourner vers la gauche : La pièce tourne dans le sens antihoraire.

L’amplitude de rotation de la pièce dépend de la taille de pas sélectionnée.

Taille de pas : Basée sur la valeur donnée pour Petit, Moyen et Grand pour les fonctions Déplacer et Tourner, la pièce sera déplacée et tournée en conséquence. L’utilisateur peut modifier la valeur en fonction de ses préférences.

Basculer : L’utilisateur peut sélectionner n’importe quelle pièce et cliquer sur l’option Basculer pour sélectionner toutes les pièces restantes. La sélection de plusieurs pièces peut être effectuée en cliquant sur les pièces respectives. La désélection peut se faire en cliquant sur les pièces sélectionnées. Les pièces peuvent ensuite être déplacées, tourner ou supprimer d’une seule manœuvre.

Supprimer : L’utilisateur peut supprimer la pièce selon la sélection. Cliquez sur OUI dans la fenêtre contextuelle pour la suppression.

Réimbrication : L’utilisateur peut utiliser l’option de réimbrication pour imbriquer toutes les pièces du layout.

Paramètres de pièce : L’utilisateur peut utiliser cette option pour modifier le format de tôle, le bord de la tôle, le sens de laminage et l’écart de pièce.

Base de données laser : L’utilisateur peut utiliser cette option pour modifier la page de données laser qui contient la matière, l’épaisseur, le diamètre de la buse, le type de gaz, le traitement et le film de protection.

Contour

Dans cette page, l’utilisateur peut sélectionner les contours, puis utiliser ces options pour les modifier.

  • Microjonction : L’utilisateur doit sélectionner le contour avant de sélectionner l’une des options listées ci-dessous :

    1. Garantir microjonction : Si un contour sélectionné possède déjà une ou plusieurs microjonctions, ne rien faire, sinon insérer une microjonction.

    2. Ajouter microjonction : Cette option ajoute plusieurs microjonctions au contour sélectionné.

    3. Réduire la microjonction : Cette option réduit la microjonction pour le contour sélectionné.

    4. Mettre microjonction en surbrillance : Cette option met en surbrillance les microjonctions dans le layout.

  • Découpe de chute : L’utilisateur doit sélectionner le contour intérieur pour effectuer les options ci-dessous :

    1. Créer une découpe de chute : Cette option est utilisée pour créer une découpe de chute.

    2. Supprimer une découpe de chute : Cette option supprime la découpe de chute sélectionnée.

  • Réglages contour : Cette option est utilisée pour modifier la largeur de microjonction, la taille de trame pour la découpe de chute et la longueur d’approche pour la découpe de chute.

Outillage

Cette option permet à l’utilisateur de déplacer l’approche vers le point cible et de modifier les paramètres d’outillage en sélectionnant le contour.

  • Déplacer l’approche : Cette option est utilisée pour déplacer l’approche en sélectionnant le point d’approche et en le déplaçant vers le nouveau point cible.

  • Découpe laser : Lors de la sélection de l’outillage, une fenêtre de dialogue s’ouvre. L’utilisateur peut alors modifier l’outillage laser.

  • L’utilisateur peut inverser le sens de la coupe : du sens horaire au sens antihoraire ou vice-versa.

  • Choisir le type de perçage en fonction de la condition requise.

  • Choisir les conditions de coupe en fonction de la condition requise.

  • Modifier les paramètres d’approche - Longueur, Rayon et Angle en fonction de la condition requise.

  • Activer/désactiver la microjonction et modifier sa largeur en fonction de la condition requise.

Squelette

Dans cette page, l’utilisateur peut déplacer et supprimer la ligne de coupe une fois que la coupe de tôles créée est activée à partir des réglages.

  • Déplacer la ligne de coupe : L’utilisateur doit activer l’option Créer une coupe de tôles pour créer la ligne de coupe, puis sélectionner la ligne de coupe à déplacer, vers la gauche ou la droite selon la distance indiquée dans la taille de pas.

  • Supprimer la ligne de coupe : L’utilisateur doit sélectionner la ligne de coupe, puis la supprimer en utilisant l’option Supprimer la ligne de coupe.

  • Réglages squelette : image::image37.jpg[Skeleton settings dialog,400]

  • Créer une coupe de tôles : Indique si le squelette doit être tranché ou non.

  • Espacement X et Y : Il s’agit de l’espacement entre les lignes de coupe respectivement verticales et horizontales.

  • Créer une tôle résiduelle : Active ou désactive la tôle entamée pour créer ou supprimer une coupe résiduelle.

  • Largeur minimale de tôle résiduelle : Même si ce qui précède est coché, une coupe finale sera ajoutée uniquement lorsque la largeur de la tôle inutilisée créée par la coupe est supérieure ou égale à cette valeur.

  • Décalage X de la coupe finale : Il s’agit de la distance par rapport à l’extrémité droite de montage de pièce et la coupe finale verticale qui crée la tôle entamée.

  • Traiter les lignes de coupe après toutes les pièces : L’activation de cette fonction lance d’abord la coupe des pièces puis de la tôle. Les paramètres suivants affectent la façon dont chaque tranche coupée individuelle est programmée :

  • Ecart de microjonction au bord de la tôle : Définir une valeur supérieure à zéro si vous souhaitez une micro-attache à gauche, aux points où la tranche coupée touche le bord de la tôle.

  • Ecart de microjonction au bord de la pièce : Définir une valeur supérieure à zéro si vous souhaitez une micro-attache à gauche, aux points où la tranche coupée touche le bord de la pièce.

  • Ecart de perçage depuis le bord de la pièce : Il s’agit de la distance du point de perçage à partir du bord de la pièce.

  • Mesurer la distance depuis le bord de la tôle : Ce paramètre identifie la distance avant le bord de la tôle où la hauteur de la tête laser est verrouillée. Le réglage de la distance est désactivé.

  • Distance de ralentissement après le bord de la feuille : Il s’agit de la distance à laquelle la tête de la machine dépasse du bord de la tôle lorsque le laser est activé mais que le réglage de la distance est désactivé.

Ordre

Cette section montre l’enchaînement de la coupe du layout.

Annuler : Toutes les modifications apportées seront perdues. Le logiciel revient à la page des programmes.

Enregistrer : Enregistre toutes les modifications.
