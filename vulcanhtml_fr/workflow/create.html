Créer une tâche

L’utilisateur doit sélectionner l’un des fichiers dxf ou plusieurs fichiers, puis sélectionner le bouton Créer une tâche. Une fois la Tâche créée, une interface avec des onglets s’affiche, comme sur la capture d’écran ci-dessous. Cette fenêtre comprend quatre onglets, comme suit :

  1. Onglet Pièces

  2. Onglet Configuration

  3. Onglet Tôles

  4. Onglet Layouts

Onglet Pièces

Dans cet onglet, l’utilisateur peut ajouter autant de pièces qu’il le souhaite. Il doit sélectionner tous les fichiers dxf, puis Créer une tâche pour ajouter toutes les pièces. Dans l’onglet Pièces, l’utilisateur peut modifier la quantité maximale, définir la priorité, le sens du grain, la rotation des pièces et le groupe. Dans cet onglet, vous pouvez gérer la liste des pièces (et leurs nombres théorique) qui doivent être imbriquées.

Onglet Configuration

Il vous permet de modifier les paramètres tels que la matière, l’épaisseur, le film de protection, le traitement et la fente de pièce.

Onglet Tôles

Il s’agit de l’onglet où vous allez gérer la liste des tôles à considérer pour l’imbrication. L’utilisateur peut ajouter des tôles de différentes tailles. Augmenter ou diminuer la quantité. La marge peut également être modifiée.

Utilisez TwinLine si vous avez besoin d’un usinage efficace de vos pièces.

Onglet Layouts

Lorsque vous sélectionnez l’onglet Layout, Vulcan Control démarre automatiquement le calcul d’imbrication si aucun layout n’est disponible. Une fois le calcul effectué, le layout calculé s’affiche sous la forme d’une liste de vignettes. Vous pouvez modifier n’importe quel layout dans la fenêtre principale Flux en cliquant sur la vignette correspondante. Un éditeur de layout complet s’ouvre, où vous pouvez modifier la position/l’orientation des pièces, modifier l’outillage, modifier l’ordre, etc.

Appuyez sur le bouton Recalculer pour imbriquer les pièces restantes (non imbriquées). Si vous souhaitez une réimbrication depuis le début, supprimez les layouts existants, puis appuyez sur ce bouton.

En appuyant sur le bouton édition, la page d’édition s’ouvre pour le layout, elle est expliquée en détail dans la section 3.3. En appuyant sur le bouton Annuler, les modifications effectuées ici ne sont pas enregistrées.

L’utilisateur peut enregistrer les layouts via enregistrer les layouts. Saisissez le nom et cliquez sur Ok pour enregistrer les layouts. Le layout enregistré sera disponible dans la page des programmes.
