Reprise

Au cas où vous avez dû arrêter un programme lors de l’exécution précédente, Vulcan Control vous permet reprendre la tôle pour enregistrer la matière.

  • "Continuer à partir du point d’arrêt" reprend le programme

  • "Redémarrer depuis le début" lance une nouvelle course

  • "Annuler" revient à l’écran précédent (pas d’exécution du programme)

La position de reprise permet à l’utilisateur de redémarrer un programme interrompu à partir d’un point spécifique. L’utilisateur peut sélectionner la ligne particulière dans l’éditeur DIN ou sélectionner directement le segment dans le dessin ou utiliser l’option Aller à. La saisie de bloc possède une option pour saisir manuellement la ligne et le numéro de bloc dans la saisie de bloc ou pour choisir le segment, une mise à jour se fait. Dans le segment sélectionné, le curseur peut être utilisé pour positionner le point exact à partir duquel le programme doit être exécuté.

Dans l’option Aller à, l’utilisateur peut naviguer entre différents contours ou segments. Après avoir sélectionné le point et cliqué sur Terminé, le mode Recherche de bloc s’active. Lorsque vous appuyez sur Démarrer, la machine se déplace vers la position à partir de laquelle le programme redémarre. Après s’être approché de la position de départ, le message suivant s’affiche : "Position cible de la recherche de bloc atteinte. Appuyer sur Démarrer pour continuer". Appuyez maintenant sur Démarrer pour commencer l’usinage.
