Nettoyage des buses et étalonnage du capteur de hauteur

L’activation de ce programme nettoie la buse au point d’effleurement de buse et étalonne le capteur de hauteur sur une plaque externe située près du point d’effleurement.

  • L’axe se déplacera vers les valeurs de décalage définies, configurées dans les réglages machine.

  • Accédez à Configuration → Réglages machine → Options API → Paramètres de commande → Nettoyage des buses et décalages d’étalonnage du capteur de hauteur.

  • En mode automatisation de fabrication, pour modifier les options de nettoyage automatique des buses, accédez à Configuration → Réglages machine → Options API → Paramètres de fonctionnement → Options de nettoyage des buses automatique.

  • Un nettoyage des buses régulier garantit une qualité constante des pièces.
