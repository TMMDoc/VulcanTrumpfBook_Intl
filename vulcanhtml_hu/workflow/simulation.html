Szimuláció futtatása

Ha részletes információt szeretne kapni arról, hogy a megmunkálási sorrend hogyan került meghatározásra, futtasson le egy szimulációt.

A kezelőfelület szimulációja

Ezen az oldalon a programhoz tartozó szimulációt futtatjuk le. A program neve és mérete a felső részen jelenik meg. Lejátszhatjuk, szüneteltethetjük és visszaállíthatjuk a szimulációt. A csúszka mutatja a szimuláció előrehaladását. Balra húzva vissza, jobbra húzva pedig előretekerhető a szimuláció.

Ez a gomb a program nagyítására szolgál.

Ezzel a gombbal a programot az eredeti pozícióba lehet visszaállítani.

Ez a gomb a program kicsinyítésére szolgál.

Ezzel a gombbal a program balra mozgatható.

Ezzel a gombbal a program jobbra mozgatható.

Ezzel a gombbal a program felfelé mozgatható.

Ezzel a gombbal a program lefelé mozgatható.

A Kész opció kiválasztásával a szimulációs ablak bezárul.

Alkatrész szimulációja

A lézerdiódával a lézer tesztüzemben azt is láthatta, hogy az alkatrész hol lesz kivágva a lemezen. Csak válassza ki az opciót, miközben az alkatrész lézeres vágási programjának próbafuttatását végzi.
