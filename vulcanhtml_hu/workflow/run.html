Programválasztó

A program kiválasztásával és a FUTTATÁS gomb megnyomásával a felhasználó a FUTTATÁS lapra navigál. A program neve, mérete, NCR állapota és a lézer paraméterei (tengelyek adatai, sebesség, gáznyomás, teljesítmény, gáztípus, fúvóka magassága, fókusztávolság, aktuális munkakoordináta-rendszer, szerszámkoordináta-rendszer, használt M funkció) a felső részen jelennek meg.

  • Start : Elindítja a programot.

  • Szünet : Szünetelteti a programot. Ez az opció csak akkor érhető el, ha a felhasználó elindítja a programot.

  • Tovább : Folytatja a szüneteltetett programot.

  • Stop vagy megszakítás : Leállítja a programot.

  • Szerkesztés : Szerkeszti a programot; A korábban említett program szerkesztési oldalára navigál.

Az aktuális aktív jelprogramhoz frissítendő időinformáció. Amint egy program indul, megkezdődik az idő számlálása; egy új program indításakor az visszaáll.

Sebesség felülbírálása

A felhasználó az Override + és Override - gombokkal befolyásolhatja a tengelymozgások sebességét (százalékban).

  • Override + : Növeli a konfigurált/beállított tengely vagy útvonal sebességének felülbírálási százalékát. 10% felett az érték 10%-os , 10% alatt pedig 1%-os lépésekben változik.

  • Override - : Csökkenti a konfigurált/beállított tengely vagy útvonal sebességének felülbírálási százalékát. 10% felett az érték 10%-os , 10% alatt pedig 1%-os dekrementumokban változik. A felhasználó a csúszkával is növelheti vagy csökkentheti a sebességet.

A felhasználó a gép működése közben is navigálhat a különböző lapok között: Lehetőség van arra, hogy a felhasználó a Futtatás lapról akkor is elmozduljon, amikor a gép éppen a lap vágásával van elfoglalva. Erre azért van szükség, mert miközben egy program fut, a kezelő esetleg a következő programcsomaggal (programszerkesztés, lézerparaméterek módosítása stb.) szeretne felkészülni a gyártásra. Ezen kívül, amikor a felhasználó a program futása közben lapot vált, egy mini lejátszó jelenik meg, amelyből megtudhatja a program aktuális állapotát, anélkül, hogy a Futtatás lapra kellene lépnie.

Kijelző

A felhasználó be- vagy kikapcsolhatja a lemez, a gép és a mozgatás megjelenítését a rajzon.

A PLC statikus üzenetei a FUTTATÁS oldal bal alsó sarkában jelennek meg.

Állapot

A felhasználó ellenőrizheti a program és a gép állapotára vonatkozó információkat; ezek csak olvashatóak.

Programvezérlések

A felhasználó a kívánt opció kiválasztásával vezérelheti a programot, amikor a program FUTTATÁS állapotban van. A programvezérlésekben kiválasztott opciók a bal alsó sarokban jelennek meg.

DIN programban megjeleníti a programvezérlők összes opcióját.
